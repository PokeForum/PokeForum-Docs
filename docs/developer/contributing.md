# 贡献指南

感谢你考虑为 PokeForum 贡献代码！

## 如何贡献

### 报告问题

如果你发现了 bug，请在 [Issues](https://github.com/PokeForum/PokeForum/issues) 中提交，并包含：

- 问题描述
- 复现步骤
- 预期行为
- 实际行为
- 环境信息

### 提交代码

1. **Fork 仓库**

\`\`\`bash
git clone https://github.com/YOUR_USERNAME/PokeForum.git
cd PokeForum
\`\`\`

2. **创建分支**

\`\`\`bash
git checkout -b feature/your-feature-name
\`\`\`

3. **提交更改**

\`\`\`bash
git commit -m "feat: add some feature"
\`\`\`

4. **推送到分支**

\`\`\`bash
git push origin feature/your-feature-name
\`\`\`

5. **创建 Pull Request**

## 代码规范

- 遵循现有代码风格
- 添加必要的测试
- 更新相关文档
- 提交信息使用 [Conventional Commits](https://www.conventionalcommits.org/) 规范
